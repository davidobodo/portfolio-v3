### PATCH

method of modifying resource where client sends parial data that is to be updated without modifying the entire resource

patch is for partial replacement
you want to update just part of the resource
a resource like a user object exists and I just want to update a single field
client transmits partial data

### PUT

method of modifying resource where client sends data that updates the entire resource

This is a complete replacement
if i want to update the entire resource
client transmits whole data

Put can be used to create a resource if the resource doesn't exist. This operation is known as UPSERT
